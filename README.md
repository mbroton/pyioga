## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This project helps to setup Google OAuth2 authentication for IMAP connection with Gmail.

If you want to read emails from Gmail in your automated E2E tests or scripts, you're in a right place!

Pyioga is an additional layer built on Google authentication libraries to make a whole process simple.

This project is built to guide you through the entire process.


### Built With

* [Python3](https://python.org/)
* [Google API Client](https://github.com/googleapis/google-api-python-client)

## Getting Started

To start reading emails from Gmail, you have to:
* Setup Google account (generate "client secrets" file)
* Use pyioga to generate token based on client secrets file


### Prerequisites

* Python3
* Gmail Account


### Google account setup

lorem

### Pyioga installation

* Install with pip

```
pip install pyioga
```

* Generate token file

```
pyioga --client-secret-file client_secrets.json --output-file token.json
```

* In application, get access token based on token.json

```
import pyioga


access_string = pyioga.get_access_token("token.json")
```
